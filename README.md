# Resume-Semantic-Search-Project

### GOAL: 
Query and Retrieve the desired profiles from the list of resumes.

### Tools used:
* Langchain
* FAISS
* Streamlit
* HuggingFaceInstructEmbeddings
* Streamlit Lottie
